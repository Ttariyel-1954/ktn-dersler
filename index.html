📚 Dərsin Məqsədi

Bu dərsdə proqramlaşdırmanın əsas anlayışları, kompüterin necə işlədiyi və ilk proqramın necə yazılacağı haqqında öyrənəcəksiniz.

🎯 Dərsin sonunda siz:
  • Proqramlaşdırmanın nə olduğunu başa düşəcəksiniz
  • Proqramlaşdırma dillərinin növlərini öyrənəcəksiniz
  • İlk sadə proqramınızı yazacaqsınız

🤔 Proqramlaşdırma Nədir?

Proqramlaşdırma - kompüterə müəyyən tapşırıqları yerinə yetirmək üçün təlimatlar verməkdir. Bu təlimatlar xüsusi dildə (proqramlaşdırma dili) yazılır.

Proqramlaşdırmanın Əsas Məfhumları:

  • Alqoritm - problemin həlli üçün addım-addım təlimatlar
  • Proqram - alqoritmin proqramlaşdırma dilində yazılmış forması
  • Kod - proqramın mətn formatında yazılışı
  • Debugging - proqramdakı səhvlərin tapılması və düzəldilməsi

💻 Proqramlaşdırma Dilləri

Müxtəlif məqsədlər üçün müxtəlif proqramlaşdırma dilləri istifadə olunur:

Populyar Proqramlaşdırma Dilləri:

  1. Python - başlanğıc üçün ideal, sadə sintaksis
  2. JavaScript - web saytlar üçün
  3. Java - böyük layihələr üçün
  4. C++ - sistemlər proqramlaşdırması
  5. C# - Windows proqramları

🚀 İlk Proqram: "Salam Dünya"

Proqramlaşdırmaya başlamaq üçün ənənəvi olaraq "Salam Dünya" proqramı yazılır:

# Python dilində print("Salam Dünya!") # JavaScript dilində console.log("Salam Dünya!"); # Java dilində public class SalamDunya { public static void main(String[] args) { System.out.println("Salam Dünya!"); } }
